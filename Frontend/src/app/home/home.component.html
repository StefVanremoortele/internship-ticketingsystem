This is a demo project in which I demonstrate a full-stack web application
written in asp.net and angular

The main goal of this project is to build a ticketing system where users can order tickets from different events

99% up-time due to load-balanced API in the cloud on Azure
Enjoy real-time updates while ordering tickets
Experience the ease of single-sign-on

Taking in consideration the heavy load a web application can get,
I had to make sure my app is concurrency and heavy performance resistant.

Everything gets logged in case of failure
