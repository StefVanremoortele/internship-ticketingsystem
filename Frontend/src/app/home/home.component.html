This is a demo project in which I demonstrate a full-stack web application,
written in asp.net as back-end and angular as front-end

The main goal of this project is to build a ticketing system where users can order tickets from different events

Taking in consideration the heavy load a web application can get,
I had to make sure my app is concurrency and heavy performance resistant. Everything gets logged in case of failure

It contains:

  • Real-time updates
  • Single Sign on onto OAuth Server
  • Load balanced API
  • Scheduled Cart timer
